Follw the steps to run the project:

1. Open cmd in root project folder
2. Write in cmd and run the command 'npm install'
3. Next run the command 'npm start '
4. Then Go to browser and write 'http://localhost:3000/log.html'
5. That's it, you can either Host a new meeting or Join in existing meeting using meeting code.
